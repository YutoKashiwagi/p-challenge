```bash
cd subject12/my-app

# React立ち上げ
yarn start

# storybook立ち上げ
yarn run storybook

# テスト実施(storybookを立ち上げておく)
yarn run test --watchAll
```

## スナップショットテストと比べてのメリット・デメリット

### メリット

- UIの変更が見やすい
- エンジニア以外でもテスト結果の確認が容易に行える

### デメリット

- 画像を使ったテストであるため、スナップショットテストと比べて不安定
- 画像のため、重い
